# Análise Empírica de Complexidade de Algoritmos

Projeto desenvolvido para a disciplina Estruturas de Dados Básicos I.

## Objetivo

O objetivo do projeto é realizar uma análise empírica da complexidade de algoritmos de busca e ordenação.

O sistema:

- executa algoritmos para diferentes tamanhos de entrada;
- mede o tempo médio de execução;
- salva resultados em arquivos CSV;
- compara os resultados com curvas assintóticas teóricas;
- detecta automaticamente a complexidade com melhor ajuste.

---

## Algoritmos Implementados

### Busca

- Busca Sequencial
- Busca Binária

### Ordenação

- Insertion Sort
- Merge Sort
